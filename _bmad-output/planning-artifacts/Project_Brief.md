# NEXUS: Project Brief

## Project Overview
NEXUS is a real estate collaboration platform initially built for REMAX Altitud to streamline communication, property matching, and agency management. It aims to scale into a multi-tenant B2B SaaS platform for regional real estate franchises and individual brokerages.

## Objective
The primary goal is to foster a network effect where agents can share buyer criteria (Searches) and other agents can propose listings (Matches), thereby accelerating deal closures. Furthermore, it provides tiered enterprise features that allow Team Leaders and Brokers to monitor deal pipelines, OKRs, and agent survival rates.

## Target Audience
1. **Every Agent:** Independent or external real estate agents who need a network to broadcast buyer needs and receive listing matches.
2. **REMAX Agents:** Premium agents within the REMAX network who get auto-matching capabilities against their office's inventory and enhanced resources.
3. **Team Leaders:** Agents managing junior associates, handling onboarding, and measuring team KPIs.
4. **Brokers:** Office owners requiring top-level visibility into agent split performance, deal flow, and overall office "Survival Line."
5. **Main Admins:** System administrators managing region/office hierarchies and subscription tiers (Free, Silver, Gold, Diamond).

## Key Value Propositions
- **Centralized Match Board:** Replaces chaotic WhatsApp groups with a structured feed of buyer "Searches" and "Matches".
- **Real-Time Data Ownership:** Brokerages maintain ownership of their internal pipeline and performance data instead of relying solely on disjointed tools.
- **Agent Survival Metrics:** Dynamic evaluation of whether an agent is in the "Green Zone" or "Red Zone" based on YTD Net Income vs. baseline targets.

## Timeline & Constraints
- The project is currently transitioning from an initial prototype/MVP into a hardened, structured software lifecycle using the **BMAD Method**.
- Development favors lean, vanilla web technologies combined with Supabase to minimize build overhead and maximize speed.
